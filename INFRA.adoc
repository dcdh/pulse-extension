== Testing

=== Stub

Stub is not a mock. A stub is a real implementation of an interface. It is used in test only to stub a dependency.

Stub must define using these annotations

[source,java]
----
    @ApplicationScoped
    @Priority(1)
    @Alternative
----

=== beans.xml

- Only defined on a test framework module
- Do not do @QuarkusTest
