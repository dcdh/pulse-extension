== debug

Launch a container in bash mode directly without running the default command.

[source,shell]
----
docker run -it --rm --entrypoint bash quay.io/debezium/kafka:3.3.1.Final
----

[source,shell]
----
docker image inspect quay.io/debezium/kafka:3.3.1.Final
----

check why container is unhealthy

[source,shell]
----
docker inspect --format='{{json .State.Health}}' <containerid>
----
